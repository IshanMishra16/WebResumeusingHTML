# Personal Web Resume (HTML Only) 
This repository contains my **personal web-based resume**, built using **pure HTML** without any external libraries or frameworks. The goal of this project is to showcase my profile, education, skills, and contact details in a clean and simple web format. 
## Project Overview - 
A single-page resume website
Created using **HTML**
 Beginner-friendly and easy to understand
 Designed to be lightweight and fast 
## Technologies Used - 
**HTML5** 
*(No CSS frameworks, JavaScript, or backend technologies used)* 
## Features
- Profile image section
- Education details
  Skills and interests
  Projects overview 
  Work experience (fresher-friendly)
- Contact information with clickable links
- Internal page navigation using anchor links
